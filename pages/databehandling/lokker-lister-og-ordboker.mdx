## While vs. for

### Sortering av ordb√∏ker med verdier

I utgangspunktetet kan ikke ordb√∏ker sorteres, s√• hvis vi for eksempel √∏nsker √• sortere n√∏klene i en ordbok etter verdier, m√• vi konvertere ordboken til en liste.

```python

   "Ravi": 39,
   "Thor": 33,
   "Martin": 34
}

# lager en liste av tupler som kan sorteres (tupler er nesten som lister, s√• du kan tenke at vi lager en liste av lister)
personer_liste = personer.items() # -> [('Ravi', 39), ('Thor', 33), ('Martin', 34)]

personer_sortert = sorted(personer_liste, key=lambda person: person[1]) # sorterer listen p√• det som ligger p√• plassering [1], alts√• verdien
print(personer_sortert) # -> [('Thor', 33), ('Martin', 34) ('Ravi', 39)]
print(personer_sortert[0][0]) # -> 'Thor'
```
