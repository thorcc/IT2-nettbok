# Plotting

## Et enkelt plott

Plotting kan enkelt gjøres ved å sende to lister til `plot`-funksjonen.
Den første listen er x-verdiene og den andre listen er y-verdiene.

> `plt.show()` gjør at programmet pauses helt til brukeren har lukket plottet, uten dette funksjonskallet ville det ikke vært mulig for brukeren å se plottet.

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.show()
```

![enkelt plott](./bilder/plotting/enkelt-plott.png)

## Akser

Akser kan defineres med `axis`-funksjonen som tar inn en liste med start og sluttverdier for aksene.

`plt.axis([x_start, x_slutt, y_start, y_slutt])`

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.axis((0, 6, 0, 20))
plt.show()
```

![akser](./bilder/plotting/akser.png)

## Tekst på akser og tittel

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.title("tittel")
plt.xlabel("tekst på x-aksen")
plt.ylabel("tekst på y-aksen")
plt.show("tittel-navn.png")
```

![tittel og navn](./bilder/plotting/tittel-navn.png)

## Flere plott i samme tegning

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16], label="x i andre")
plt.plot([1, 2, 3, 4], [1, 8, 27, 64], label="x i tredje")
plt.plot([1, 2, 3, 4], [1, 16, 81, 256], label="$x^4$") # tekst inne i dollartegn tolkes som matte
plt.legend() # viser en oversikt over farger og tilhørende labels
plt.title("tittel")
plt.xlabel("tekst på x-aksen")
plt.ylabel("tekst på y-aksen")
plt.show()
```

## Stiler

Det er mulig å sette på stiler, de defineres ved å legge til en string etter listene som skal plottes.
Stilen skrives på formen `'[markør][linje][farge]'`

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16], marker="o", color="g", linestyle="-")
plt.show()
```

![stiler](./bilder/plotting/stiler-o-g.png)

Hvis man for eksempel ikke ønsker streker, dropper man linjevalget `-` i stilen.

```python
import matplotlib.pyplot as plt

plt.plot([1, 2, 3, 4], [1, 4, 9, 16], marker="o", color="g")
plt.show()
```

### Markører

> Det finnes mange flere markører, sjekk [dokumentasjonen til matplotlib](https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.plot.html#examples-using-matplotlib-pyplot-plot) for en oversikt over alle.

| Tegn   | Beskrivelse    |
| ------ | -------------- |
| `'.'`  | point          |
| `','`  | pixel          |
| `'o'`  | circle         |
| `'s'`  | sqaure         |
| `'v'`  | triangle down  |
| `'p'`  | pentagon       |
| `'\*'` | star           |
| `'x'`  | x              |
| `'X'`  | filled x       |
| `'d'`  | diamond        |
| `'D'`  | filled diamond |

### Linjer

| Tegn   | Beskrivelse |
| ------ | ----------- |
| `'-'`  | solid line  |
| `'--'` | dashed line |
| `'-.'` | dash-dot    |
| `':'`  | dotted      |

### Farger

| Tegn  | Beskrivelse |
| ----- | ----------- |
| `'r'` | red         |
| `'g'` | green       |
| `'b'` | blue        |
| `'c'` | cyan        |
| `'m'` | magenta     |
| `'y'` | yellow      |
| `'k'` | black       |
| `'w'` | white       |

## Eksempel: Ordbok med verdier

`bar`-plott er fine når man vil plotte ordbøker med verdier.
De lages med `plt.bar(x_verdier, y_verdier)`.

```python
import matplotlib.pyplot as plt

personer = {
   "Ravi": 41,
   "Thor": 33,
   "Martin": 34
}

navn = list(personer.keys())
alder = list(personer.values())
plt.bar(navn, alder)
plt.title("Alder på lærere")
plt.ylabel("år")
plt.show()
```
